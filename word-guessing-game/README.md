# Word Guessing Game

A simple word guessing game implemented in Python.

## Description

The objective of the game is to guess a hidden word by suggesting letters within a certain number of attempts. The game provides feedback on correct and incorrect guesses, and displays the current state of the word after each guess.

## Features
- Randomly selects a word for each game session
- Tracks correct and incorrect guesses
- Limits the number of incorrect attempts
- Input validation to ensure only single letters are accepted
- Console interface

## How to Play
1. Run the `main.py` file.
2. Guess the word by entering one letter at a time.
3. If the guessed letter is in the word, it will be revealed in its correct position(s).
4. If the guessed letter is not in the word, you lose an attempt.
5. The game ends when you guess the word or run out of attempts.

## Requirements
- Python 3.x

## Running the Game
Open a terminal in the game folder and run:

```
python main.py
```
